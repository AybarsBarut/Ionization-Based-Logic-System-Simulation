# Security and Safe Use

## Scope

This repository is an educational software simulation. It is not intended to
control laboratory equipment, power supplies, gas systems, vacuum systems, or
high-voltage hardware.

## Reporting Software Vulnerabilities

Please report potential software vulnerabilities privately through GitHub
Security Advisories when available. Do not include credentials, tokens, or
private datasets in a public issue.

## Scientific Safety

The included parameters and outputs must not be interpreted as:

- Safe laboratory operating limits
- Hardware design recommendations
- Validated plasma chemistry data
- Radiation or nuclear-process predictions
- Instructions for constructing experimental apparatus

Use the project only for conceptual simulation, software education, and
method-development exercises.
