# Changelog

## 1.0.0 - 2026-06-07

- Added simplified ionization, recombination, excitation, emission, and
  conductivity models.
- Added electron, positive-ion, negative-ion, and metastable populations.
- Added adaptive ODE integration and mean electron-energy dynamics.
- Added threshold logic gates, half adder, full adder, Schmitt trigger, and SR
  latch examples.
- Added Monte Carlo uncertainty propagation and coefficient calibration.
- Added one-dimensional reaction-diffusion simulation.
- Added reproducible CSV and graphical outputs.
- Added automated test coverage and GitHub Actions workflow.
